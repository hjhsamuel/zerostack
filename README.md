# zerostack
A Go microservices framework based on [gin](https://github.com/gin-gonic/gin)

## Usage

How to install zerostack:

```shell
go install github.com/hjhsamuel/zerostack/zs
```

The usage of zerostack:

```shell
zs --help
```
